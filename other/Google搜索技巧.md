# 搜索

+ 搜索效率

  谷歌 > 公众号 > 短视频 > 百度

+ ""限定关键词 

  在关键词两边加上英文的双引号 "AAA"

+ intitle 限定标题

  在关键词前面加上intitle:A

+ allintitle 限定标题多个关键词

  allintitle:AAA BBB

+ intext 限定文章内容关键词（主要用于搜索个人资料）

  intext:AAA

+ inurl 限定网址关键词

  inurl:cctv

+ site 限定网址来源

  site:zhihu.com

+ imagesize 限定图片尺寸

  imagesize:2560*1440

+ filetype 限定文件格式 (了解xx行业,就搜索这个行业的研究报告）

  母婴行业2023 filetype:pdf
